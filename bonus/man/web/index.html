Builtins prototypes

Echo

Display a line of text

Usage : echo [-params] {string1} {string2} ...

Parameters :
-n : do not output the trailing newline



Setenv

Insert or modify an environment variable

Usage : setenv {variable name} {varialbe content}



Unsetenv

Delete an environment variable

Usage : unsetenv {variable name}



env

Display all environment variables

Usage : env



where

Print the location of a binary

Usage : where {binary}



whoami

Print effective userid

Usage : whoami



cd

Change the working directory

Usage : cd {directory | option}

option :
- : Changes to the previous working directory



which

Shows the full path of (shell) commands.

Usage : which {command}



set

Set or unset options and positional parameters

Usage : set {command}



unset

Unset local variable

Usage : unset {variable}



alias

Define or display aliases

Usage : alias {alias name} {command}



history

Display the history

Usage : history



jobs

Display status of jobs in the current session

Usage : jobs



fg

Run job in the foreground

Usage : fd {job id}



export

Set the export attribute for variables

Usage : export {variable name}



sort

Usage: sort [OPTION]... [FILE]

With no FILE, read standard input.

Ordering options:
-f, fold lower case to upper case characters
-i, consider only printable characters
-n, compare according to string numerical value
-r, reverse the result of comparisons



yes

Display a string repeatedly until killed

Usage : yes {string}



Exit

Cause normal process termination

Usage : exit {return value}



PWD

Print name of current/working directory

Usage : pwd


Other prototypes


|| and &&

OR and AND

Exemple : echo "Folder" && ls || pwd



Multiples pipes

Exemple : ls | cat -e | wc -l



Redirection

Type : > >> < <<

Exemple : ls > file



Parenthesis

Exemple : (ls && ls ; ls | wc -c) | cat -e



Environment variable

Exemple : echo $PATH, echo $?, echo ${PATH}, MyPath:${PATH} on $pwd



~

Home directory of a user

Exemple : ~ => home user, ~root => home root



Magic quotes

Exemple : yes `ls`



Globbing

Exemple : echo *.c, it will print all file that finish by '.c'



Background

Execute a command in background

Exemple : yes "run in background" &



PS1

Custom prompt string

Option :
\u, \v, $?, \w, \l, \t

Exemple : set PS1='$> \033[31m[\u]\033[0m \033[32m\W\033[0m [\l] [\t] >
That give : $>[username] directory [nb_line] [time] >



Scripting

Custom executable file

Exemple :

#!42sh
echo "curent directory : " `pwd`
echo "Param 1 is : " $1



Local alias

Exemple :
alias cwdcmd ls => execute ls at every change of directory
alias postcmd pwd => execute pwd after every command
alias precmd date => Print the date before the prompt
set ignoreeof => ignore CTRL-D



History

Show and interact with command history


Option :
!! : Execute last command
!x : Execute the command number x in history
!command : Execute the last command that start with 'command'
!-x : Execute the last x command, (!-1 for last, ect ...)

Exemple : !ls, !-1, !2, !!
